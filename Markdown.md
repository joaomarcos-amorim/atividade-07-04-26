# Atividade

# O que é `margin` no CSS?
O `margin` no CSS define o **espaço externo** ao redor de um elemento, ou seja, a distância entre ele e outros elementos da página.

## 🧠 Para que serve?  
  
O `margin` é usado para controlar o **espaçamento entre elementos**.  
 ex:
- **margin-top:** 10px;  
- **margin-right:** 20px;  
- **margin-bottom:** 15px;  
- **margin-left:** 5px;

# 📘 O que é `padding` no CSS?
O `padding` no CSS define o **espaço interno** de um elemento, ou seja, a distância entre o conteúdo e a borda.

## 🧠 Para que serve?  
O **`padding`** serve principalmente para **controlar o espaço interno de um elemento**, ou seja, a distância entre o conteúdo (texto, imagem, botão etc.) e a borda do elemento. Ele é usado para **tornar o conteúdo mais legível e visualmente agradável**, sem “grudar” na borda.
ex:
- **padding-top:** 10px; 
- **padding-right: 20px;** → espaço interno (o foco aqui)  
- **padding-bottom: 15px;** → borda  
- **padding-left: 5px;** → espaço externo
